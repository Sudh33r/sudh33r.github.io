Jira Cloud Migration Assistant supports project-by-project migration from Jira Server to Jira Cloud and Jira Service Management with greater flexibility, reliability, and simplicity. Significantly cut down migration time by moving the data you need at the time that works best for you.

Use the Jira Cloud Migration Assistant to migrate:

  • Projects, users, and groups from Jira Software, Jira Service Management, and Jira Work Management (formerly Jira Core) on Server to Cloud.

  • Jira Service Management customer accounts on Server to Cloud.

  • Advanced Roadmaps plans with linked issue sources (projects and single-project boards).

Get insights into the progress of your migrations

The Jira Cloud Migration Assistant includes app assessment. Auditing apps on your Server, assessing them, then installing apps on your Cloud site before migration removes the risk of your apps blocking your migration at a crucial time. These tools are helpful to provide a better understanding of what apps you’re running and what your team is using to help you make an informed decision when considering cloud migration.




Check out the documentation to learn when to use the app. You’ll need a Jira Cloud site to begin migrating with the Jira Cloud Migration Assistant. If you haven’t already, contact us to start a free extended cloud trial. The app requires that your Jira Server site runs on version 7.6 or above.
